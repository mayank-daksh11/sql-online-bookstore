# 📊 Online Bookstore SQL Analysis

---

## 📌 Project Overview
This project focuses on analyzing an Online Bookstore database using SQL. The dataset is structured into multiple relational tables including Books, Customers, and Orders. The goal of this project is to extract meaningful business insights such as revenue trends, customer behavior, product performance, and inventory status.

---

## 🛠 Tools & Technologies
- PostgreSQL  
- SQL (Structured Query Language)  

---

## 🗂 Database Structure
The database is designed using a relational model:

- **Books** → Stores product details  
- **Customers** → Stores customer information  
- **Orders** → Stores transaction data  

These tables are connected using foreign keys to perform relational analysis.

---

## 📊 Key Analysis Performed

### 🔹 Sales & Revenue Analysis
- Total revenue generated from orders  
- Monthly sales trends  
- Revenue contribution by genre  

### 🔹 Customer Analysis
- Top spending customers  
- Repeat customers (frequent buyers)  
- City-wise customer distribution  

### 🔹 Product Analysis
- Best-selling books  
- Most popular genres  
- Author-wise sales performance  

### 🔹 Inventory Analysis
- Remaining stock after sales  
- Low stock identification  

---

## 📈 Key Insights
- Identified top-performing books and genres  
- Found high revenue-generating cities  
- Analyzed customer purchasing patterns  
- Tracked inventory usage and remaining stock
